================================================================================
                              QUICK REFERENCE: WHAT EXISTS
================================================================================

Database Schema (shared/schema.ts):
- consignmentSubmissions: seller vehicle submissions
- inventoryCars: approved vehicles for sale (has createdAt for days-on-lot)
- buyerInquiries: inquiry tracking with inventoryCarId link
- siteSettings: branding, contact, commission, GHL config
- users: admin authentication
- vehicleAlerts: customer notification preferences
- appointments: scheduled appointments
- tradeInRequests: trade-in submissions
- statusHistory: consignment status changes

Key API Routes (server/routes.ts):
- /api/inventory (GET, POST, PATCH, DELETE)
- /api/consignments (GET, POST, PATCH status)
- /api/inquiries (GET, POST, PATCH status)
- /api/settings (GET, PATCH)
- /api/auth/* (login, logout, session, setup)
- /api/seller/* (portal auth)
- /api/vin-decode/:vin
- /api/vehicle-makes, /api/vehicle-models

Admin Pages (client/src/pages/admin/):
- Dashboard.tsx
- Inventory.tsx
- Leads.tsx
- Consignments.tsx
- Settings.tsx
